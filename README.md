# Super ToDo Planner

Современное приложение-планировщик задач на Python и PyQt5.

## Что реализовано

- Расширенный менеджер задач с поддержкой проектов, тегов и оценки времени
- Мощный фильтр по статусу, проекту, тегам и текстовому поиску
- Быстрый импорт/экспорт в JSON и CSV
- Тёмная и светлая тема
- Интерактивная таблица задач с сортировкой и деталями
- Расширенные фильтры по диапазону дат, сегодня/неделя/просроченные
- Контекстное меню и горячие клавиши для быстрого редактирования
- Сохранение темы, фильтров и окна между запусками
- Статистика по общему количеству, выполненным, просроченным и задачам высокого приоритета
- Хранение данных в SQLite + автоматическое резервное JSON-хранилище

## Запуск

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Запустите приложение:

```bash
python main.py
```

## Файлы

- `main.py` — графический интерфейс и логика управления задачами
- `database.py` — расширенный слой работы с SQLite и импорт/экспорт
- `styles.qss` — стиль приложения
- `test_database.py` — базовые тесты для слоя данных

## Тесты

Запустите базовые проверки:

```bash
python test_database.py
```

## Упаковка

Можно собрать один исполняемый файл через `pyinstaller`:

```bash
pip install pyinstaller
pyinstaller --onefile main.py
```
